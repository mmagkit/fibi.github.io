HealthyBite – Multi‑page HTML/CSS site (Responsive, Accessible, Animated)

Files:
- index.html        Home with hero + featured recipes
- recipes.html      Searchable recipe list (client-side)
- calorie.html      Calorie tracker with live total
- tips.html         Health tips grid
- contact.html      Contact form with inline feedback
- style.css         Global styles (responsive + animations + a11y)
- script.js         Interactivity (menu, save-toast, search, tracker)

How to run (Visual Studio / VS Code):
1) Open the HealthyBite folder in your editor.
2) Right‑click index.html and choose “Open With Live Server” (VS Code) OR open in your browser.
3) Navigate between pages via the top nav.
